准备 前期准备
  1. 视频直播要求较高，请选择至少i5以上电脑配置
  2. 网络确保高速，最好独占
  3. 使用前关闭360等软件，防止干扰误删
  4. 使用前关闭其它软件和浏览器，以免占用资源，引起卡顿

Q 没有声音怎么办？
检查下麦克风是否开启
检查麦克风音量
检查耳机是不是有问题
若有多个麦克风选项，可换另一个麦克风试试

Q 摄像头没启动或有问题怎么办？
检查摄像头是否连接正确
在设置页面的摄像头面板中，重新检测 试试是否正常
外接式摄像头可重新插拔下试试
重启软件尝试
重启电脑尝试
不要点击重启按钮，而是将电脑关闭一会，再重新打开尝试

Q 意外崩溃怎么办？
若能关闭软件，则关闭软件；若不能关闭，则重启软件
重启电脑，查看是否正常

Q 直播出错怎么办？
查看网络是否正常，网络抖动是否严重
暂停直播，然后点击 继续直播 看是否正常
重启软件，查看是否正常

Q 连麦注意事项
连麦者最好不要在一个屋子，避免声音重采样，造成回声
连麦时，声音源建议选择 麦克风，因 系统声音源 会有回声
